# drawai

Web-приложение для превращения детских каракулей в осмысленные 3D-изображения и видео.

## Описание

`drawai` — это прототип платформы, разработанной для местного института, позволяющей детям создавать простые наброски (каракул) и преобразовывать их в 3D-модель или видео с помощью удалённого Gradio API-модели, а также генерировать «игрушечные» версии при наличии человека на изображении посредством детектора YOLOv8.

Основные компоненты:

* **Flask** для веб-сервера и обработки запросов.
* **gradio-client** для взаимодействия с моделью `linoyts/sketch-to-3d`.
* **Ultralytics YOLOv8** (`yolov8m.pt`) для обнаружения человека перед генерацией игрушечной фигуры.

## Структура репозитория

```
.
├─ app.py                        # Основной серверный код (Flask + Gradio + YOLO)
├─ lolkek.py                     # Шаблонный скрипт для математической генерации изображений (формулы)
├─ yolov8m.pt (не включён)       # Весы модели YOLOv8m (скачайте вручную)
├─ temp/                         # Временные файлы для обработки эскизов
├─ static/
│  └─ results/                   # Сгенерированные изображения и видео
└─ templates/
   ├─ index.html                 # Интерфейс с холстом для рисования и формой генерации
   └─ result.html                # Страница результатов: отображение 3D-модели и видео
```

## Зависимости

* Python 3.8 или выше
* Flask
* gradio-client
* ultralytics (YOLOv8)

## Установка

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/HellDiver830/drawai.git
   cd drawai
   ```
2. Создайте и активируйте виртуальное окружение (рекомендуется):

   ```bash
   python -m venv venv
   source venv/bin/activate   # Linux/macOS
   venv\\Scripts\\activate  # Windows
   ```
3. Установите необходимые пакеты:

   ```bash
   pip install flask gradio-client ultralytics
   ```
4. **Скачайте веса YOLOv8m** и поместите файл `yolov8m.pt` в корень проекта (рядом с `app.py`):

   ```bash
   wget https://huggingface.co/Ultralytics/YOLOv8/resolve/main/yolov8m.pt
   ```

## Запуск приложения

```bash
python app.py
```

Откройте в браузере: `http://localhost:5000`.
